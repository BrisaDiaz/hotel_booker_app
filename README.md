## Environment variables

- APP_SECRET= Jwt secret key
- HOST=Appication hots name
- BACKEND_URL=HOST/api/graphql
- DATABASE_URL=Postgresql database connection string

- CLOUDINARY_NAME
- CLOUDINARY_API_KEY
- CLOUDINARY_API_SECRET
