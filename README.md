## üïπSet up project

#### 1. Install my-project with npm

```bash
  npm install my-project

  cd my-project
```

#### 2. Create and seed the database

```bash
npx prisma migrate dev --name init

npx prisma db seed

```

#### 3. Start the app

```bash
  npm run dev

```
